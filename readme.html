คู่มือการใช้งาน TABRadio Add-on for NVDA

ภาพรวม:

TABRadio เป็น Add-on สำหรับ NVDA ที่ช่วยให้ผู้ใช้สามารถฟังวิทยุออนไลน์และทีวีออนไลน์ได้ง่ายๆ ผ่านปุ่มลัด โดยสามารถบันทึกสถานะการฟังล่าสุด (resume) เพื่อกลับมาฟังต่อได้แม้ปิด NVDA ไปแล้ว

ปุ่มลัดหลัก:

NVDA+Shift+Space    : เปิด/ปิดการทำงานของ TABRadio
NVDA+Shift+F1: เล่นโหมดทีวี
NVDA+Shift+F2: เล่นโหมดวิทยุ
NVDA+Alt+Space      : ประกาศว่ากำลังฟังช่องอะไรอยู่
NVDA+Shift+F4       : เพิ่มเสียง (+10%)
NVDA+Shift+F3       : ลดเสียง (-10%)
NVDA+Shift+Right    : ไปยังช่องถัดไป
NVDA+Shift+Left     : ไปยังช่องก่อนหน้า

วิธีใช้งาน:

  1. เปิด TABRadio: กด NVDA+Shift+Space เพื่อเริ่มทำงาน หากเคยฟังช่องใดไว้ก่อนปิด NVDA จะกลับมาฟังช่องนั้นทันที
  2. สลับโหมด วิทยุ/ทีวี: กด NVDA+Shift+F1/F2 เพื่อสลับโหมด ระบบจะจำโหมดล่าสุดไว้
  3. เลือกช่อง: NVDA+Shift+Right Arrow เพื่อไปช่องถัดไป, NVDA+Shift+Left Arrow เพื่อย้อนกลับช่องก่อนหน้า
  4. ปรับระดับเสียง: เพิ่มเสียง NVDA+Shift+F4, ลดเสียง NVDA+Shift+F3
  5. ฟังสถานะปัจจุบัน: กด NVDA+Alt+Space เพื่อฟังชื่อช่องที่กำลังเล่นอยู่

หมายเหตุ:

  • TABRadio จะบันทึกช่อง, ระดับเสียง, และโหมดล่าสุดไว้ใน state.json
  • ต้องมี VLC หรือ FFplay ติดตั้งในเครื่อง
  • ใช้ได้กับ Windows และ Linux

User Guide for TABRadio Add-on for NVDA

Overview:

TABRadio is an NVDA add-on that allows you to listen to online radio and TV streams using keyboard shortcuts. It saves your last playback state (resume) so you can continue where you left off even after restarting NVDA.

Main Hotkeys:

NVDA+Shift+Space    : Toggle TABRadio on/off
NVDA+Shift+F1/F2      : Switch between Radio and TV mode
NVDA+Alt+Space      : Announce the currently playing channel
NVDA+Shift+F4       : Increase volume (+10%)
NVDA+Shift+F3       : Decrease volume (-10%)
NVDA+Shift+Right    : Next channel
NVDA+Shift+Left     : Previous channel

How to Use:

  1. Start TABRadio: Press NVDA+Shift+Space to activate the add-on. If you were listening to a channel before closing NVDA, it will automatically resume.
  2. Switch Radio/TV Mode: Press NVDA+Shift+F1 or F2 to toggle between radio and TV mode. The last mode will be remembered.
  3. Select Channels: Next channel - NVDA+Shift+Right Arrow; Previous channel - NVDA+Shift+Left Arrow
  4. Adjust Volume: Increase volume - NVDA+Shift+F4; Decrease volume - NVDA+Shift+F3
  5. Announce Current Status: Press NVDA+Alt+Space to hear the name of the channel currently playing

Notes:

  • TABRadio saves current channel, volume level, and mode in state.json
  • Requires VLC or FFplay installed
  • Compatible with Windows and Linux
